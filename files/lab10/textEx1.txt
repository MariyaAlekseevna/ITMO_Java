Java — это объектно-ориентированный язык, основанный на C ++, с сильной поддержкой сетевой, многопоточной и
компонентно-ориентированной разработки.
Первая версия JDK ( Java Development Kit ) 1.0, выпущенная в 1995 году, установила первый крупный публичный
стандарт Java с синтаксисом на C ++ и иерархией классов, включая поддержку сетей, потоков, обработку событий,
обработку исключений, многопоточность и Разработка графического интерфейса с помощью набора абстрактных оконных
средств (AWT). Вскоре после этого были введены классы Java Database Connectivity (JDBC) для обмена данными с
базами данных.
